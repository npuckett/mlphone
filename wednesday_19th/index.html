Wednesday 19th - Classes & localStorage

Core Concepts

CLASSES: Organize related data and behavior
         ↓
    Encapsulation (hide complexity)
    Reusability (use in multiple projects)
    Maintainability (easier to modify)
         ↓
    Clean, professional code organization

LOCALSTORAGE: Persist data between sessions
         ↓
    Save state (character health, stress, etc.)
    Load state (restore previous session)
    Clear state (reset to defaults)

This session demonstrates object-oriented programming with classes and data persistence with localStorage, showing how these concepts simplify complex code and create more sophisticated interactions.
